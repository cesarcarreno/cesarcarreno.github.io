Project descriptions!

PacMen Exercise:
This project creates a new pacman after pressing Add PacMan. Start Game makes the PacMan move accross the screen. Additionally pressing Add PacMan will add another one and pressing Start Game will make them move faster. The working link can be found in the navigation bar and the repository here: https://github.com/cesarcarreno/PacMen_Exercise

Eye Movement Exercise:
This project makes both eyes follow the mouse movements throughout the screen. It uses primarily the onmousemove event and it tracks the x & y position of the cursor. The working link can be found in the navigation bar and the repository here: https://github.com/cesarcarreno/Eye-Exercise

Real Time Bus Tracker Exercise:
It shows the bus stops between MIT and Cambridge's most reviewed restaurant "Russel House Tavern". The project was originally showing the bus stops between MIT and Harvard. The improvements are that the initial marker now starts over the restaurant and after the button is pressed than it animates the bus stops to the new location. The working link can be found in the navigation bar and the repository here: https://github.com/cesarcarreno/The-Real-Time-Bus-Tracker
